## ⚡ Quick Start

### Installation

```bash
# 1. Repository klonen
git clone https://github.com/arthurfischer009/python-learning-project.git
cd python-learning-project

# 2. Virtual Environment aktivieren
source .venv/bin/activate

# 3. Dependencies installieren (falls nötig)
pip install -r requirements.txt
```

### Projekte ausprobieren

```bash
# Trading Bot
python trading-bots/Training/Simple_Traiding_Bot.py

# Portfolio Allocation
python portfolio-tools/portfolio_allocation.py

# Solana Dashboard
streamlit run solana-portfolio/portfolio_dashboard.py
```
